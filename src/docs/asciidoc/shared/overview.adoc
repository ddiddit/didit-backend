== Overview

=== 인증

모든 인증이 필요한 API는 요청 헤더에 Bearer 토큰을 포함해야 합니다.

[source,http]
----
Authorization: Bearer {accessToken}
----

토큰이 없거나 유효하지 않은 경우 401 응답을 반환합니다.
토큰이 만료된 경우 `/api/v1/auth/refresh` 를 통해 재발급 받아야 합니다.

=== 에러 코드

|===
| 코드 | HTTP 상태 | 설명

| INVALID_REQUEST
| 400
| 잘못된 요청

| INTERNAL_SERVER_ERROR
| 500
| 서버 내부 오류

| INVALID_REFRESH_TOKEN
| 401
| 유효하지 않은 리프레시 토큰

| EXPIRED_REFRESH_TOKEN
| 401
| 만료된 리프레시 토큰

| WITHDRAWN_USER
| 403
| 탈퇴한 회원

| UNSUPPORTED_OAUTH_PROVIDER
| 400
| 지원하지 않는 소셜 로그인

| OAUTH_USER_INFO_FAILED
| 502
| 소셜 로그인 사용자 정보 조회 실패

| USER_NOT_FOUND
| 404
| 사용자를 찾을 수 없음

| USER_CONSENT_NOT_FOUND
| 404
| 약관 동의 정보를 찾을 수 없음

| DUPLICATE_NICKNAME
| 409
| 이미 사용 중인 닉네임

| NOTIFICATION_SETTING_NOT_FOUND
| 404
| 알림 설정을 찾을 수 없음
|===
